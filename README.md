# Salon Management System

A Java-based Salon Management System developed for the Quality Assurance and Cloud Computing project.

## Overview

The Salon Management System is designed to help manage the daily operations of a salon. It allows users to manage customers, employees, salon services, appointments, and payments while demonstrating Object-Oriented Programming (OOP) principles in Java.

## Features

* Customer management
* Employee management
* Salon service management
* Appointment booking
* Payment processing
* Unit testing with JUnit 5

## Technologies Used

* Java
* Maven
* JUnit 5
* Git
* GitHub
